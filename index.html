Welcome to the AI Assistant Library!

This website showcases a collection of AI assistant configurations created by Daniel Rosehill between 2024 and 2025.

Browse through the configurations using the sidebar navigation or search for specific ones using the search box.

Each configuration includes:

  • A descriptive name
  • A detailed description
  • The system prompt used to configure the AI assistant

You can easily copy the system prompts to your clipboard or download the entire configuration as a markdown file.

Select a configuration from the sidebar to get started!
